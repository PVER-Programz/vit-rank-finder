# VIT Rank Finder 📡🏆

AI was able to make this stuff. We are so doomed.

## 🌟 Features

- **📡 Batch Scanner**: Automatically scan a range of registration numbers within a specific batch.
- **🏆 Live Leaderboard**: Real-time ranking of students based on CGPA after scanning.
- **📊 Detailed View**: Embedded access to comprehensive grade details via [vit-grade.vercel.app](https://vit-grade.vercel.app).
- **✨ Smart Hints**: Dynamic registration number previews as you configure your scan range.
- **🎨 Premium UI**: Modern dark-mode interface with glassmorphism effects and responsive design.

## 🚀 How It Works

1. **Configure**: Enter a target registration number (e.g., `24BVD1077`). The app automatically extracts the batch prefix.
2. **Range Selection**: Set the `Start Number` and `End Number` for the scan.
3. **Initiate**: Click **INITIATE SCAN**. The engine will query the batch data securely.
4. **Analyze**: Once complete, view the **Leaderboard** to see where the target student stands relative to the batch.

## 🛠️ Technical Details

- **Frontend**: Vanilla HTML5, CSS3, and JavaScript (ES6+).
- **Styling**: Custom CSS variables, Flexbox/Grid layouts, and Outfit typography.
- **API**: Powered by the [vit-grade.onrender.com](https://vit-grade.onrender.com) backend API.
- **Responsiveness**: Fully optimized for desktop and mobile views.
